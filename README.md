## Description
- It was a great way to practise how to pass props from one component to another and update the value depending on the user input.
- The user can be able to: 
   - Add a todo 
   - Edit the todo added
   - Remove the todo
   - Make the todo as complete


#### Technologies
- Html
- Css
- React
